# Design Guidance

The site should feel:
- calm
- breathable
- low-friction
- dark-friendly
- emotionally grounded
- non-aggressive
- human

Avoid:
- excessive animation
- dashboard energy
- productivity obsession
- overstimulation
- startup hype culture

The goal is trust and orientation.
Not spectacle.

---

# Permanent Public Boundary

The public site must never expose:
- protected continuity systems
- internal orchestration
- sensitive operational architecture
- private memory structures
- credentials
- Rock-core context
- secret sauce internals

Public-facing surfaces are windows.
Not the vault.
